Tento projekt je ukázkou automatizovaného testování webové aplikace pomocí Pythonu, Selenium WebDriveru a frameworku PyTest.

Cílem projektu bylo navrhnout jednoduchý, ale přehledně strukturovaný testovací framework využívající Page Object Model a pytest fixtures.

Projekt simuluje reálné uživatelské scénáře v demo e-shopu SauceDemo a ověřuje klíčovou funkcionalitu aplikace.


Použité technologie
- Python  
- Selenium WebDriver  
- PyTest  
- webdriver-manager
